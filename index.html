🪟 Windows

Windows, Microsoft tarafından geliştirilen dünyanın en popüler işletim sistemlerinden biridir. Kullanımı kolaydır ve oyun, ofis programları ile günlük kullanım için oldukça uygundur. Windows 11 en güncel sürümlerden biridir ve milyonlarca bilgisayarda kullanılmaktadır.

🐧 Linux

Linux, açık kaynak kodlu ve ücretsiz bir işletim sistemidir. Güvenliği, hızı ve kararlılığı ile bilinir. Yazılımcılar, sunucular ve siber güvenlik uzmanları tarafından sıkça tercih edilir. Ubuntu, Debian, Fedora ve Arch Linux en popüler Linux dağıtımlarıdır.

⚖️ Windows ve Linux Karşılaştırması

Windows kullanım kolaylığı ve oyun desteğiyle öne çıkarken, Linux yüksek performansı, güvenliği ve özelleştirilebilir yapısıyla dikkat çeker. Windows ücretli lisans gerektirebilir, Linux ise çoğunlukla ücretsizdir.

💻 Programlama Nedir?

Programlama, bilgisayara belirli görevleri yaptırmak için komut yazma işlemidir. Yazılan komutlara "kod", bunların tamamına ise "program" denir.

⌨️ İlk Kod

print("Merhaba Dünya!")

Bu kod ekrana "Merhaba Dünya!" yazdırır. Birçok programlama dilinde ilk örnek olarak kullanılır.

📝 Değişkenler

isim = "Ahmet"
yas = 15

Değişkenler veri saklamak için kullanılır. Metin, sayı ve daha birçok veri türü değişkenlerde tutulabilir.

➕ Operatörler

5 + 3 = 8
10 - 4 = 6
6 * 2 = 12
20 / 4 = 5

Operatörler matematiksel işlemler yapmak için kullanılır.

🔁 Döngüler

for i in range(5):
    print(i)

Döngüler aynı işlemi tekrar tekrar çalıştırmak için kullanılır.

❓ If-Else

yas = 18

if yas >= 18:
    print("Ehliyet alabilir.")
else:
    print("Ehliyet alamaz.")

If-Else yapısı koşullara göre farklı kodların çalışmasını sağlar.

📦 Fonksiyonlar

def selam():
    print("Merhaba!")

selam()

Fonksiyonlar aynı kodu tekrar yazmadan kullanmayı sağlar.

🌐 HTML

<h1>Merhaba Dünya</h1>

HTML web sayfalarının iskeletini oluşturur.

🎨 CSS

h1{
 color:red;
}

CSS web sitelerinin tasarımını ve görünümünü düzenler.

⚡ JavaScript

alert("Merhaba!");

JavaScript web sayfalarına hareket ve etkileşim kazandırır.
